운영체제 (Operating System)
Windows, iOS, Android, Linux, Unix...
시스템(하드웨어, 소프트웨어)의 자원을 효율적으로 관리하는 소프트웨어이다.

실행중인 프로그램을 프로세스로 관리한다.
PID(Process ID) 관리 코드를 부여하여 관리.

운영체제는 골고루 자원을 할당해 주어 멀티 프로세스를 생성하여 처리한다.

메신저는 채팅 작업을 하면서 동시에 파일 전송 작업을 수행하기도 하므로
하나의 프로세스 내에서 멀티 태스킹 작업을 할 수 있도록 구현할 수 있다
즉 하나의 ㅡㅍ로세스가 두 가지 이상의 작업을 처리할 수 있는 이유는 멀티 스레드가 있기 때문이다.

스레드(Thread) => 실타래 : 코드의 실행 흐름을 의미한다.
프로세스 내에 스레드가 두 개라면 두개의 코드 실행 흐름이 생긴다는 의미.

멀티 프로세스는 OS 내에 프로세스가 여러개.
프로세스는 서로 독립적이므로 하나의 프로세스에서 오류가 발생해도 다른 프로그램에 영향을 끼치지 않는다.
ex) 한글파일과 엑셀파일은 독립적임.


멀티 스레드는 프로세스 내에 스레드가 여러개.
멀티 스레드는 프로세스 내부에서 생성되기 때문에 하나의 스레드가 예외를 발생시키면 프로세스가 종료되므로 다른 스레드에 영향을 미친다.
ex) 카카오톡(프로세스) 내의 스레드(선물하기, 채팅보내기) 중 하나가 장애 발생하면 카카오톡 자체가 종료.
==> 예외처리가 매우 중요!!
